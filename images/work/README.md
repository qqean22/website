# Work Portfolio Images

This folder contains images for the portfolio/work page.

## Image Naming Convention

Name your images as follows:
- `project-1.jpg` (or `.png`, `.webp`)
- `project-2.jpg`
- `project-3.jpg`
- etc.

## Recommended Image Specifications

- **Format**: JPG, PNG, or WebP
- **Aspect Ratio**: Square (1:1) recommended for best grid display
- **Resolution**: Minimum 1200x1200px for high-quality display
- **File Size**: Optimize images to keep file sizes reasonable (under 500KB per image recommended)

## Adding Images

1. Add your images to this folder following the naming convention above
2. The work.html page will automatically display them in the grid
3. Images will be automatically cropped to fit the square grid format

## Current Placeholders

The work page currently has 9 placeholder slots. You can:
- Add up to 9 images (project-1.jpg through project-9.jpg)
- Add more by duplicating the work-item structure in work.html
- Remove placeholders you don't need
