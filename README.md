# Creative Studio — Website

Static site for a creative studio. Safe to deploy to any static host (Netlify, Vercel, GitHub Pages, etc.).

## Deploy

- **Netlify:** Connect the repo; build command and publish directory can stay empty (static site).
- **Vercel:** Connect the repo; no build step required.
- **GitHub Pages:** Enable Pages for the repo and choose the branch; root is the site.

Security headers are set via `_headers` (Netlify) and `vercel.json` (Vercel). Replace contact placeholders on the Contact page with your details before or after going live.
